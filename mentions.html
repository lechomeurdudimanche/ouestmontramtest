Mentions Légales

Présentation

Où est mon tram ? est une application web indépendante et non officielle. Elle n'est ni éditée ni affiliée au SMMAG ou à M TAG. Elle exploite les API publiques ouvertes pour informer en temps réel sur les transports en commun de l'aire grenobloise.

Crédits techniques

Cette application utilise les API publiques de Mobilités M (le service d'information des mobilités de l'aire grenobloise).

Les données de transport sont fournies par le SMMAG (Syndicat Mixte des Mobilités de l'Aire Grenobloise) et sont diffusées sous la licence Open Database License (ODbL).

Les données de calcul d'itinéraire sont également fournies par l'API Mobilités M via OpenTripPlanner.

Les données sont à titre indicatif.

Les cartes affichées dans l'application utilisent :

  • Les tuiles cartographiques de CARTO
  • Les données géographiques d'OpenStreetMap, diffusées sous licence ODbL
  • La bibliothèque de cartographie open source Leaflet

La recherche d'adresses et de lieux utilise deux services :

  • La Base Adresse Nationale, un service public de l'État français
  • Photon, un moteur de géocodage open source basé sur les données OpenStreetMap, développé par Komoot

Aucune donnée saisie dans la recherche n'est conservée par ces services au-delà du temps de la requête.

Librairies open source : Leaflet, QRCode.js, Google Fonts

Hébergement : Github

Confidentialité & Cookies

Cette application a été conçue dans le respect de votre vie privée :

  • Aucune donnée personnelle n'est collectée, stockée côté serveur ou revendue.
  • Aucun traceur publicitaire ou outil d'analyse intrusif n'est utilisé.
  • L'historique de recherches et les adresses favorites de la page Itinéraire sont sauvegardés uniquement sur votre appareil (via le stockage local du navigateur). Ces données ne quittent jamais votre appareil et peuvent être effacées en vidant le cache de votre navigateur. Même chose pour vos arrêts favoris sur la page Horaires.
  • La géolocalisation de votre appareil peut être utilisée pour centrer la carte ou définir un point de départ. Elle est activée uniquement à votre demande, n'est jamais transmise à un serveur tiers, et reste soumise aux autorisations accordées dans votre navigateur.
